# WoolBrawl — Paper 1.21.4 плагин

Battlebox-style режим "Шерстяная потасовка" с GUI выбора кита.

## Как собрать `.jar`

Я не смог собрать плагин на своей стороне — нет сети, Paper API не скачался.
Ты соберёшь его у себя. Нужен **JDK 21** и **Maven** (или IntelliJ IDEA).

### Вариант A: IntelliJ IDEA (проще всего)

1. Открой IntelliJ → `Open…` → выбери папку проекта (там где `pom.xml`)
2. IDEA автоматически подтянет зависимости (несколько минут)
3. Справа снизу — панель Maven → `Lifecycle` → двойной клик на `package`
4. Готовый jar будет в `target/WoolBrawl-1.0.0.jar`
5. Кинь его в `plugins/` своего Paper сервера

### Вариант B: Maven в терминале

```
mvn package
```

Jar появится в `target/WoolBrawl-1.0.0.jar`.

## Использование

1. Поставь плагин, перезагрузи сервер — создастся `plugins/WoolBrawl/config.yml`
2. Правь координаты в конфиге ИЛИ используй команды:
   - `/wb setpoint lobby` — встань где нужно лобби, выполни
   - `/wb setpoint spawn-red` — спавн красных
   - `/wb setpoint spawn-blue` — спавн синих
   - `/wb setpoint spectator` — точка зрителей
   - `/wb setzone min` + `/wb setzone max` — углы зоны захвата 3x3
   - `/wb setarena min` + `/wb setarena max` — углы арены (где ТНТ работает)
3. `/wb reload` — применить
4. `/wb show` — посмотреть текущий конфиг
5. `/wb start` — запустить игру
6. `/wb stop` — остановить

## Игровой цикл

1. **Lobby (20с)** — все в точке лобби, в центральном слоте hotbar **изумруд "Выбор кита"**.
   ПКМ по изумруду → открывается GUI 3×9 с 9 китами из пула (рандом из 13 китов каждый раз).
   Клик по киту = выбор, подсветится "✓ ВЫБРАН".
2. **Playing** — команды автобалансятся, каждый получает:
   - Выбранный кит (если не выбрал — первый из пула)
   - Кожаная броня красится в цвет команды
   - **Слот 7 (8-й по счёту)**: 64× шерсть цвета команды
   - **Слот 8 (9-й по счёту)**: ножницы
   - 3 сек стан с резистом перед "GO!"
   - Центр под чёрной неломаемой шерстью первые 15с, потом белая
3. **Захват** — ломаешь шерсть в зоне → на её место шерсть твоего цвета.
   **Шерсть в центре не дропается** (нельзя фармить).
4. **Победа** — вся зона твоего цвета, вражеская команда истреблена, или больше очков по таймеру.

## Важное

- **ТНТ ломает только зону захвата**. Где бы ты его не поставил — взрыв повредит только блоки в зоне. Снаружи — разлетится без повреждений.
- **Свои своих не бьют** (friendly fire off).
- **Голод заморожен** на 20 во время игры.
- **Конфиг-мир** должен существовать. По умолчанию `world`.

## Структура

```
pom.xml
src/main/
  resources/
    plugin.yml
    config.yml
  java/net/cubo/woolbrawl/
    WoolBrawlPlugin.java       — главный класс
    config/WbConfig.java        — загрузка config.yml
    kit/
      Kit.java                  — один кит
      KitRegistry.java          — все 13 китов
      KitSelectGUI.java         — GUI окно выбора
    game/
      GameState.java            — enum состояний
      Team.java                 — enum команд
      PlayerData.java           — данные игрока в раунде
      GameManager.java          — мозг: старт/стоп/тики/победа
    arena/
      ZoneManager.java          — работа с зоной захвата
    listener/
      InteractListener.java     — ПКМ по изумруду → GUI
      KitSelectListener.java    — клик в GUI → выбор кита
      BreakListener.java        — ломание шерсти = захват, без дропа
      DeathListener.java        — смерть = в зрители
      TntListener.java          — ТНТ только в зоне захвата
      MiscListener.java         — friendly fire, голод, выход
    command/
      WbCommand.java            — /wb ...
```

## Известные моменты

- GUI открывается только во время фазы **LOBBY**. Если нажать изумруд в другое время — ничего не произойдёт.
- Если плагин выключить во время игры — сработает `onDisable` и всё аккуратно свернётся.
- Для истинно-рабочего подсчёта очков/побед команд надо было бы хранить это в базе, но я не делал — по ТЗ не было.
